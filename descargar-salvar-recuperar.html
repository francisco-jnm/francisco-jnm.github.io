Descargar ficheros desde un sitio web

Aunque existen funciones para leer directamente ficheros desde una URL, da muchos menos problemas descargar el fichero a local y luego leerlo. Si se trata de un fichero plano (csv, html, xml, txt, etc…), es muy directo descargarlo utilizando la función “download.file”.

En este ejemplo, nos interesa un fichero csv que está en la url “https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2Fss06hid.csv” y que queremos guardar con el nombre “idaho_housing.csv”. También queremos comprobar si ya lo hemos descargado previamente para no descargarlo en ese caso

file_name <- "idaho_housing.csv" 
file_url <- "https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2Fss06hid.csv"
if(!file.exists(file_name)) {download.file(file_url, destfile = file_name)}

El fichero descargado se puede eliminar directamente desde R con la función “unlink”.

unlink("idaho_housing.csv")

Si se trata de un fichero comprimido (fichero “zip”, generalmente), es lo mismo que con un fichero plano salvo que, además, realizamos la descompresión con la función “unzip”:

file_name <- "hospital_compare.zip" 
file_url <- "https://d396qusza40orc.cloudfront.net/rprog%2Fdata%2FProgAssignment3-data.zip" 
if(!file.exists(file_name)) {
    download.file(file_url, destfile = file_name)
    unzip(file_name, exdir = ".") }

# Eliminamos lo descargado
unlink("hospital_compare.zip")
unlink("Hospital_Revised_Flatfiles.pdf")
unlink("hospital-data.csv")
unlink("outcome-of-care-measures.csv")

Cuando el fichero no es plano (una hoja Excel, por ejemplo), utilizamos “download.file” en modo binario (mode=“wb”). En este ejemplo, nos interesa una hoja excel xlsx:

file_url <- "https://d396qusza40orc.cloudfront.net/getdata%2Fdata%2FDATA.gov_NGAP.xlsx"
file_name <- "060_natural_gas.xlsx"
if(!file.exists(file_name)) {download.file(file_url, destfile = file_name, mode="wb")}

# Eliminamos lo descargado
unlink("060_natural_gas.xlsx")

Salvar/recuperar un marco de datos en formato estándar de R

La manera mas simple de salvar y recuperar más tarde un marco de datos es utilizar el formato estándar de R (“.rds”). Para ello, sólo hay que utilizar la función “saveRDS” para salvar y “readRDS” para recuperar:

ciudades = data.frame(
    ciudad = c("Albacete", "Alicante", "Almería"),
    miles_habitantes = c(390.31, 1838.97, 700.71)
)
print(ciudades)
##     ciudad miles_habitantes
## 1 Albacete           390.31
## 2 Alicante          1838.97
## 3  Almería           700.71
#Salvar a fichero RDS
saveRDS(object = ciudades, file = "ciudades.rds")

#Leer (recuperar) de fichero RDS
ciudades = readRDS(file = "ciudades.rds")

# Eliminamos rds
unlink("ciudades.rds")

Salvar/recuperar un marco de datos en fichero csv

Para guardar nuestro marco de datos de manera que se pueda abrir o importar en otras aplicaciones, el formato rds no es la mejor opción. Es mucho mas flexible salvarlo a un fichero “.csv” mediante la función “write.table”:

write.table(x = ciudades, 
            file = "ciudades.csv", 
            sep = ";",
            dec = ".",
            quote = FALSE,
            row.names = FALSE)
  • x: indica el nombre del marco de datos a salvar
  • file: indica el nombre del fichero de texto en el que vamos a salvar el marco de datos
  • sep: indica el carácter que separa las columnas en el fichero de texto
  • dec: indica el símbolo que actua como separador decimal. Casi siempre será o bien el punto o bien la coma.
  • quote = FALSE: indica que no se utilicen caracteres especiales (como comillas o comillas dobles) para rodear los campos de tipo carácter
  • row.names = FALSE: indica que no se incluye en el fichero el número de cada línea

Se puede echar un vistazo a las primeras líneas de un fichero plano con la función “readLines”. El parámetro “n” indica cuántas filas queremos leer como máximo.

readLines("ciudades.csv", n=4)
## [1] "ciudad;miles_habitantes" "Albacete;390.31"        
## [3] "Alicante;1838.97"        "Almería;700.71"

La recuperación de un marco de datos salvado a fichero csv se hace con la función “read.table”:

ciudades <- read.table(
    file = "ciudades.csv",
    header = TRUE,
    sep = ";",
    dec = ".",
    na.strings = "",
    stringsAsFactors = FALSE,
    encoding = 'UTF-8'
)

print(ciudades)
##       ciudad miles_habitantes
## 1   Albacete           390.31
## 2   Alicante          1838.97
## 3 Almer<ed>a           700.71
# Eliminamos csv
unlink("ciudades.csv")
  • file: indica el nombre del fichero de texto del que vamos a cargar el marco de datos
  • header = TRUE. Esto indica que la primera fila del fichero contiene los nombre de las columnas. Suele ser lo habitual pero, por defecto, R considera lo contrario.
  • sep: indica el carácter que separa las columnas en el fichero de texto
  • dec: indica el símbolo que actua como separador decimal. Casi siempre será o bien el punto o bien la coma.
  • na.strings: indica los caracteres que corresponden a un valor “perdido”. Depende de la aplicación que generó el fichero. Si se generó con R será “NA”, con excel sería “#N/A”, en otros casos puede ser un espacio en blanco. Incluso se puede especificar que pueden ser varios. Por ejemplo: na.strings = c(“NA”, “#N/A”, " ")
  • stringsAsFactors = FALSE. Esto se debe incluir porque R hereda de sus primeras versiones la “manía” de transformar los campos de tipo caracter a otro tipo llamado “factor” que viene a ser como un “entero etiquetado”. Mejor evitarlo.
  • encoding: indica el código de caracteres activo cuando se salvó el fichero (se indica en las opciones de R)
