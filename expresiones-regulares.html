Las expresiones regulares son los métodos más poderosos para filtrar y transformar datos pero cuesta un poco acostumbrarse a ellas. En la práctica, suele ser suficiente con dos funciones:

  • “grepl” para buscar un patrón en una cadena de caracteres
  • “gsub” para sustituir un patrón por otro en una cadena de caracteres

La función grepl (encontrar)

El uso más simple de “grepl” sería para averiguar si una palabra está en una cadena. “grepl” devuelve TRUE o FALSE según la palabra esté o no.

grepl(pattern = "Panza",
      x = c("Don Quijote", "Sancho Panza"))
## [1] FALSE  TRUE

Las opciones de búsqueda son inacabables.

El “gorrito” busca sólo al principio de la palabra.

grepl(pattern = "^Don",
      x = c("Don Quijote", "Es Don Quijote"))
## [1]  TRUE FALSE

El “dolar” busca sólo al final de la palabra

grepl(pattern = "mar$",
      x = c("El mar", "El mar mediterraneo"))
## [1]  TRUE FALSE

El “+” busca repeticiones de la parte del patrón que va entre paréntesis. Los paréntesis son ignorados por “grepl”.

grepl(pattern = "y b",    x = "muy              bien")
## [1] FALSE
grepl(pattern = "y( )+b", x = "muy              bien")
## [1] TRUE

La “doble barra” seguida de una “d” indica un número del 0 al 9.

grepl(pattern = "\\d", x = c("1 de enero", "ONU", "???"))
## [1]  TRUE FALSE FALSE

La “doble barra” seguida de una “w” indica un número del 0 al 9, una letra (minúscula o mayúscula) o un guión bajo.

grepl(pattern = "\\w", x = c("1 de enero", "ONU", "???"))
## [1]  TRUE  TRUE FALSE

El “punto” vale por cualquier carácter.

grepl(pattern = ".", x = c("1 de enero", "ONU", "???"))
## [1] TRUE TRUE TRUE

La “doble barra” seguida de un “s” indica un espacio, tabulador o fin de línea

grepl(pattern = "\\s", x = 
"Hola Adios")
## [1] TRUE

La “doble barra” seguida de un “n” indica fin de línea

grepl(pattern = "\\n", x = 
"Hola 
Adios")
## [1] TRUE

El “corchete” es como un “o”, vale cualquiera de los elementos que contiene.

grepl(pattern = "b[ou]", x = "muy bien")
## [1] FALSE
grepl(pattern = "b[ou]", x = "muy bonito")
## [1] TRUE
grepl(pattern = "b[ou]", x = "muy bueno")
## [1] TRUE

Dentro del “corchete”, el “guión” permite definir una secuencia:

grepl(pattern = "[a-d]", x = c("badajoz", "toronto")) #letras a, b, c, d
## [1]  TRUE FALSE

Dentro del “corchete”, el “gorrito” es la negación:

grepl(pattern = "[^a-z]", x = c("badajoz", "Badajoz")) #no minúsculas
## [1] FALSE  TRUE

La función gsub (sustituir)

cadena <- "      :: C/ Castilla, 28001, Madrid, España.   "

Eliminar espacios a principios de palabra:

gsub(pattern = "^\\s+", replacement = "", x = cadena)
## [1] ":: C/ Castilla, 28001, Madrid, España.   "

Eliminar espacios a final de palabra:

gsub(pattern = "\\s+$", replacement = "", x = cadena)
## [1] "      :: C/ Castilla, 28001, Madrid, España."

Eliminar las letras (minúsculas, mayúsculas y la ñ):

gsub(pattern = "[a-zA-Zñ]+", replacement = "", x = cadena)
## [1] "      :: / , 28001, , .   "

Eliminar números:

gsub(pattern = "\\d", replacement = "", x = cadena)
## [1] "      :: C/ Castilla, , Madrid, España.   "

Eliminar todo menos losnúmeros:

gsub(pattern = "[^0-9]", replacement = "", x = cadena)
## [1] "28001"

Esta es la funcionalidad mas potente (aunque es todo un arte): Particionar una cadena en trozos y “jugar” con los trozos. Los trozos se identifican mediante una doble barra y un número de orden.

En este caso, extraemos el nombre de la calle que es el segundo de los cuatro trozos de la partición:

gsub(pattern = "(^\\s+::\\s)(.+)(,\\s[0-9])(.+)$", 
     replacement = "\\2", 
     x=cadena)
## [1] "C/ Castilla"

Filtrar un marco de datos con grepl

provincias <- read.table(
   file = "provincias.csv",
   header = TRUE,
   sep = ";",
   dec = ".",
   na.strings = "",
   stringsAsFactors = FALSE,
   encoding = 'UTF-8'
)

library(dplyr)

Provincias cuyo nombre contiene un espacio:

provincias %>%
   select(provincia) %>%
   filter(grepl(pattern = "\\s", x = provincia))
##                provincia
## 1            Ciudad Real
## 2              La Coruña
## 3         Islas Baleares
## 4             Las Palmas
## 5 Santa Cruz de Tenerife

Regiones cuyo nombre empieza por vocal:

provincias  %>%
   group_by(region) %>%
   summarize(total = length(provincias)) %>%
   select(region) %>%
   filter(grepl(pattern = "^[AEIOU]", x = region))
## # A tibble: 6 x 1
##   region        
##   <chr>         
## 1 Andalucía     
## 2 Aragón        
## 3 Asturias      
## 4 Extremadura   
## 5 Islas Baleares
## 6 Islas Canarias
