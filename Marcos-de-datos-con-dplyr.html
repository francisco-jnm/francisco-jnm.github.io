Paquete DPLYR

Para manejar un marco de datos, uno de los paquetes más utilizados es “dplyr”, que aporta funciones bastante similares a las del lenguaje SQL:

  • select
  • filter
  • arrange
  • rename
  • mutate
  • group_by / summarize"
library(dplyr)

df_estadisticas <- read.table(file = "02 R Programming_050_Marcos_De_Datos.csv", 
                            header = TRUE, 
                            sep = ";", 
                            dec = ".",  
                            na.strings = "NA",
                            stringsAsFactors = FALSE)

GROUP BY AND SUMMARIZE

Las funciones “group_by” and “summarize” permite calcular campos totalizados a partir de campos agrupados.

head( 
  
  summarize(group_by(df_estadisticas, equipo), 
                    suma = sum(puntos),
                    minimo = min(puntos),
                    cuartil1 = quantile(puntos, probs=0.25),
                    mediana = median(puntos),
                    cuartil3 = quantile(puntos, probs=0.75),
                    maximo = max(puntos),
                    recuento = length(puntos)

LA TUBERÍA %>%

El operador %>% (que se lee “then” o “y después”) permite enlazar varias de las funciones anteriores, de manera que no tenemos que ir sobreescribiendo un marco de datos auxiliar hasta llegar al definitivo.

df_estadisticas %>% 
    select (jugador, equipo, partidos, puntos) %>%
    mutate(puntos_por_partido = round((puntos / partidos),2) ) %>%
    filter(equipo == "FNB") %>%
    arrange(desc(puntos_por_partido))
