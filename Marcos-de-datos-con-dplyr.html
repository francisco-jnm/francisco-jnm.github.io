Paquete DPLYR

Para manejar un marco de datos, uno de los paquetes más utilizados es “dplyr”, que aporta funciones bastante similares a las del lenguaje SQL:

  • select
  • filter
  • arrange
  • rename
  • mutate
  • group_by / summarize
library(dplyr)

provincias = read.table(
  file = "provincias.csv",
  header = TRUE,
  sep = ";",
  dec = ",",
  na.strings = "NA",
  stringsAsFactors = FALSE,
  encoding = 'UTF-8'
)

SELECT

Con la función “select” podemos indicar los campos que queremos mostrar:

head ( select (provincias, provincia, habitantes, pib_per_capita) )
##   provincia habitantes pib_per_capita
## 1   Almería     700712          20465
## 2     Cádiz    1240434          17343
## 3   Córdoba     787819          17964
## 4   Granada     916486          17894
## 5    Huelva     521605          19185
## 6      Jaén     638991          17465

También podemos seleccionar los campos que no queremos mostrar mediante el signo “-”:

head( select (provincias, -pib, -pib_per_capita) )
##      region provincia habitantes superficie
## 1 Andalucía   Almería     700712       8775

GROUP BY AND SUMMARIZE

Las funciones “group_by” and “summarize” permite calcular campos totalizados a partir de campos agrupados.

head( 
  
  summarize(group_by(provincias, region), 
                    total_habitantes = sum(habitantes),
                    media_habitantes = mean(habitantes)
                    )
  
      )

LA TUBERÍA %>%

El operador %>% (que se lee “then” o “y después”) permite enlazar varias de las funciones anteriores, de manera que no tenemos que ir sobreescribiendo un marco de datos auxiliar hasta llegar al definitivo.

provincias %>%
  mutate("Millones de habitantes" =
           round(habitantes / 1000000, 2)) %>%
  filter(region == "Andalucía")  %>%
  select (provincia, superficie, "Millones de habitantes") %>%
  arrange(desc(superficie))
##   provincia superficie Millones de habitantes
## 1   Sevilla      14036                   1.94
## 2   Córdoba      13771                   0.79
## 3      Jaén      13496                   0.64
## 4   Granada      12646                   0.92
## 5    Huelva      10127                   0.52
## 6   Almería       8775                   0.70
## 7     Cádiz       7440                   1.24
## 8    Málaga       7306                   1.65
